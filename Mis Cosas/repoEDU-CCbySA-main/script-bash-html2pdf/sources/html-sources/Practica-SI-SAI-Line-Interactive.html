    Práctica
     
  1. VirtualBox: Arrancar unha máquina virtual Windows amd64:
       
    1. Conectar o SAI á rede electrica e encendelo.
    2. Conectar o SAI mediante USB ao PC anfitrión.
       
    3. Acceder á configuración da máquina virtual arrancada e agregar o dispositivo(SAI) USB conectado.
    4. Lanzar navegador e dirixirse á URL [1]
       
    5. Dirixirse a Software e descargar o software de monitorización[5]
    6. Acceder ao arquivo descargado e proceder á sua instalación.
       
    7. Lanzar navegador e dirixirse á URL http://localhost:15178/ViewPower
