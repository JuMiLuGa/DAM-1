    Práctica
     
  1. Instalar xestor de correo Thunderbird:
       
    • setxkbmap es #Cambiar o mapa de teclado ao idioma español.
    • sudo su - #Acceder á consola de root(administrador) a través dos permisos configurados co comando sudo (/etc/sudoers, visudo)
      • apt update #Actualizar o listado de paquetes dos repositorios (/etc/apt/sources.list, /etc/apt/sources.list.d/)
      • apt search thunderbird #Buscar calquera paquete que coincida co patrón de búsqueda thunderbird
      • apt -y install thunderbird thunderbird-l10n-gl #Instalar o paquete thunderbird(xestor de correo) e o paquete thunderbird-l10n-gl(idioma galego para o xestor de correo thunderbird). Co parámetro -y automaticamente asumimos yes a calquera pregunta que ocorra na instalación do paquete.
      • exit #Saír da consola do usuario root, para voltar á consola do usuario kali sen permisos de root
  2. Configurar Thunderbird
     
    • thunderbird & #Executar thunderbird en segundo plano, devolveńdose o prompt da consola
      1. Cubrir os campos do formulario solicitado para configurar a conta de correo, e picar en Continue:
      2. Picar en Done
      3. Acceder a gmail
      4. Permitir acceso a thunderbird

      5. Cambiar o idioma da interface a Galego: Preferences ➝ Buscar language ➝ Galician ➝ Cambiado
      6. Abrir o xestor claves PGP en thunderbird: Ferramentas ➝ OpenPGP Key Manager
      7. Crear un novo par de claves: Generate ➝ New Key Pair ➝ Generate Key ➝ Confirm
      8. Configurar a conta para permitir cifrar(clave pública destinatario), descifrar(clave privada propia) e asinar correos(clave privada propia): Configuración da conta ➝ End-To-End Encryption ➝
  3. Enviar correo cifrado/asinado: Escribir ➝ Seguranza ➝ Escoller a/s opción/s que interesen
